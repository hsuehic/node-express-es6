# Nodejs Express ES6 Application Template

A template for Nodejs Express ES6 Application

## Features

- ES6 supported
- Hot reload
- Express

## Get Involved
1. get the template
```
    git clone 
```
1.  install gulp
```JavaScript
    npm i -g gulp
```
1. install nodemon
``` JavaScript
    npm i -g nodemon
```

1. start
```JavaScript
    npm run start
```
1. build
```JavaScript
    npm run build
```
1. debug
```JavaScript
    npm run debug
```
